Welcome to ShapeLearner Prediction API


Here you will find the API - User Guide.

Getting the N-Most Likely Classes.

How to get the N-Most Likely classes for a given partID ?

Please request the following address : http://IPShapeLearnerAPI:ServerPort/predict/id/<idPart>/nmax/<nMax> (Click for an example : idPart = 4, nMax = 2)
The API output is a JSON-file formated as following:

[{"predictedClasses": [["Bielle", 0.31891118093705084], ["Verin", 0.24939257330154962]], "partID": 4}]

Getting Every Classes with a likehood >= P%.

How to get all the classes with a likelihood greater or eqal to pMax (in pourcent) for a given partID ?.

Please request the following address : http://IPShapeLearnerAPI:ServerPort/predict/id/<idPart>/pmax/<pMax> (Click for an example : idPart = 6, pMax = 10)
The API output is a JSON-file formated as following:

[{"predictedClasses": [["Bielle", 0.35996145231206178], ["Vis", 0.14678096880781383], ["Axe", 0.10750174180390874]], "partID": 6}]

Getting the N-most likely classes with a likehood >= P%.

How to get the N-Most Likely classes with a likelihood greater or eqal to pMax (in pourcent) for a given partID ?.

Please request the following address : http://IPShapeLearnerAPI:ServerPort/predict/id/<idPart>/nmax/<nMax>/pmax/<pMax>
(Click for an example : idPart = 5, nmax = 4, pMax = 2)
The API output is a JSON-file formated as following:

[{"predictedClasses": [["Vilebrequin", 0.1596721664389377], ["Bielle", 0.14759627500270478], ["Moteur Electrique", 0.14754412256093191], ["Motoreducteur", 0.079397147019835773]], "partID": 5}]

How to Recompute the Knowledge Base.

In order to recompute the Knowledge Base please access to this address : http://IPShapeLearnerAPI:ServerPort/recomputeModel.
Please be patient, the process may take a while.
